1. **What is the core problem-solving methodology used in this programming introduction?**
   A. A waterfall approach with analysis, design, implementation, and maintenance
   B. A six-step approach divided into “Thinking” (steps 1–4) and “Coding” (steps 5–6)
   C. An agile sprint cycle with planning, development, review, and retrospective
   D. A top-down decomposition into modules, functions, and classes
   **Answer: B**

2. **Which of the following is the correct sequence of the six steps?**
   A. Identify inputs/outputs → Understand problem → Write pseudocode → Test → Convert → Work by hand
   B. Understand problem → Identify inputs/outputs → Work by hand → Write pseudocode → Convert to program → Test
   C. Write pseudocode → Understand problem → Identify inputs/outputs → Work by hand → Test → Convert to program
   D. Test → Understand problem → Identify inputs/outputs → Convert to program → Write pseudocode → Work by hand
   **Answer: B**

3. **Which of these is *not* a key component of the “Thinking” phase?**
