### Prompt Engineering Principles for Learning
1. **Start with "Help me understand..." not "Write code that..."**
2. **Mention your constraints:** "Using only concepts from Week 1..."
3. **Ask for explanations:** "Explain each line"
4. **Request appropriate complexity:** "Keep it simple, no functions or classes"
5. **Seek patterns:** "What pattern does this follow?"
6. **Verify understanding:** "Is my understanding correct that...?"
