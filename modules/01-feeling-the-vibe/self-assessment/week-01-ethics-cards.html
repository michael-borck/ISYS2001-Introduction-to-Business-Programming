Programming Ethics: Real Dilemmas

Click each card to explore ethical scenarios you might face

1
The Deadline Dilemma

Your temperature converter is due in 2 hours. You understand the concepts but keep getting syntax errors. Your friend offers their working code.
Copy the code and submit it
Use AI to fix your syntax errors
Look at friend's code for patterns, then fix yours

Discussion Points:

  • What's the difference between copying and learning from examples?
  • How can deadlines affect ethical decisions?
  • What are the long-term consequences of each choice?
2
The AI Temptation

You're struggling with the concept of type conversion. AI could explain it perfectly, or it could just write the whole program for you.
Ask AI to write the complete program
Ask AI to explain type conversion with examples
Use AI to check if your understanding is correct

Discussion Points:

  • What's the purpose of assignments - getting code or learning?
  • How can AI enhance vs. replace learning?
  • What skills are you building with each approach?
3
The Group Project

Your team is building a calculator. One member used AI to generate advanced code none of you understand, but it works perfectly.
Submit it - it works, that's what matters
Ask the team to simplify it together
Rewrite it using only concepts everyone understands

Discussion Points:

  • What's more important - working code or understood code?
  • How does this impact the team's learning?
  • What happens in the real world with unmaintainable code?
4
The Tutorial Trap

You found a perfect YouTube tutorial that builds exactly what you need. You could follow along and submit, or use it to understand concepts.
Copy the tutorial code exactly
Watch to understand, then build your own version
Use tutorial concepts but different example

Discussion Points:

  • When does learning from resources become copying?
  • How can you make tutorial knowledge your own?
  • What's the value of struggling vs. following along?
5
The Understanding Test

Your code works but you're not sure why. The assignment requires explaining your code. You could ask AI to explain it for you.
Submit AI's explanation as your own
Use AI to help you understand, then write your own
Trace through code manually until you understand

Discussion Points:

  • What's the difference between AI's words and your understanding?
  • How can you verify you truly understand something?
  • Why might instructors ask for explanations?
6
The Peer Pressure

Everyone in class is using AI to complete assignments quickly. You're taking longer because you're trying to understand. You feel left behind.
Start using AI like everyone else
Continue your approach but feel stressed
Find a balance - use AI to accelerate learning

Discussion Points:

  • Is faster always better in learning?
  • What are you optimizing for - grades or knowledge?
  • How can you use AI without compromising learning?
