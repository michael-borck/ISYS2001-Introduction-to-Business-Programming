# ISYS2001-Introduction-to-Business-Programming

This repository contains curriculum materials, programming exercises, and resources for the ISYS2001 Introduction to Business Programming course at Curtin University. The course provides foundational programming skills specifically tailored for business applications and contexts.

## Overview

This course introduces students to programming concepts and practices essential for modern business environments. Materials are primarily delivered through interactive Jupyter Notebooks, combining theoretical concepts with hands-on programming exercises.

## Prerequisites

- Basic computer literacy
- Fundamental understanding of business concepts
- No prior programming experience required

## Contents

The repository is organized into the following sections:

- **`modules/`** - Course modules containing lecture materials, tutorials, and interactive programming exercises
- **`assessment/`** - Assessment materials, assignments, and project specifications
- **`other-support/`** - Additional resources, reference materials, and supplementary content

## Usage

### For Students

1. Clone or download the repository to access course materials
2. Ensure you have Jupyter Notebook or JupyterLab installed
3. Navigate to the relevant module folder for your current week/topic
4. Open the notebook files (.ipynb) to view content and complete exercises

### For Instructors

The materials are structured to support weekly delivery with progressive skill building. Each module contains both instructional content and practical exercises suitable for in-class activities or assignments.

## Technical Requirements

- Python 3.x
- Jupyter Notebook or JupyterLab
- Standard Python libraries for business applications (pandas, matplotlib, etc.)
